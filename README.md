# 🛍️ Prathyeka Creations – Shopify E‑Commerce Store

This is a fully functional **Shopify eCommerce store** built for the handmade brand **Prathyeka Creations**.  
The store showcases handcrafted products including resin art, jewelry, thread bangles, hand-painted sarees, and custom mobile pouches.

🔗 **Live Store Preview:**  
👉 https://prathyeka-creations.myshopify.com/  
🔑 *Password: prathyeka* (if store is password protected)

---

## ✨ Features

- 🧵 Handmade product catalog (Resin Art, Jewelry, Bangles, Pouches, Sarees)
- 🖼️ Product collections and categories
- 🛒 Shopping cart and checkout system
- 💵 Cash on Delivery (COD) payment option
- 📱 Fully responsive design (works on mobile + desktop)
- 🎨 Custom branding with logo, colors, and typography
- 📸 Real product images from business inventory

---

## 🛠️ Technologies Used

- **Shopify** – E‑commerce platform
- **Dawn Theme** – Clean and modern Shopify theme
- **Liquid** – Shopify templating language
- **HTML/CSS** – Custom styling and layout
- **GitHub** – Project documentation and portfolio
